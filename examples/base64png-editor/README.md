# Base64 PNG Editor

You can read [here](https://blog.kie.org/2020/10/kogito-tooling-examples%e2%80%8a-%e2%80%8ahow-to-create-a-custom-editor-in-a-react-application.html) a step-by-step tutorial of how this custom Editor was built.

This package is a Custom Editor made with React, which enables you to edit a `.base64png` file.

A `.base64png` file is a PNG image converted to base64 without its headers (e.g. data:image/png;base64).
