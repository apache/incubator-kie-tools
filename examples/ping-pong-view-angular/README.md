# Ping-Pong View :: Angular implementation

The Ping-Pong View is an interface that components can implement to be used inside a Ping-Pong View Envelope.

This package provides a Ping-Pong View implementation as a Angular application.

It's build into 3 projects:

- An Angular application that can run in an iFrame;
- A Web Component from the angular components that can be rendered into any page;
- A lib used to help render the Web Component in a container.
