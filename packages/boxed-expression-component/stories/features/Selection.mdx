# Selection

The boxed expression component header and all cells have the access to the Selection context.
It gives the option to "Copy", "Cut", "Paste" and "Reset" a boxed expression. A feature that
comes with it is the possibility to select multiple cells using the mouse or the keyboard.
The selection customizes the cell navigation, and filter on which cell will performed an action.
